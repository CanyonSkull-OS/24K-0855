
*PseudoCode*:
```
INPUT num1, num2, op
SWITCH(op)
  CASE("+"):
    Ans = num1 + num2
    OUTPUT Ans
    BREAK
  CASE("-"):
    Ans = num1 - num2
    OUTPUT Ans
    BREAK
  CASE("*"):
    Ans = num1 * num2
    OUTPUT Ans
    BREAK
  CASE("/"):
    Ans = num1 / num2
    OUTPUT Ans
    BREAK
```
